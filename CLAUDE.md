# Bio-Well EU Shopify Theme

## Project Overview

This is a Shopify theme for the Bio-Well EU store, built on the **Concept v5.3.2** theme by RoarTheme. Bio-Well sells bioenergy measurement devices and accessories (cameras, sensors, filter packs, calibration units, etc.) to a European and international audience.

## Theme Architecture

- **Pure Shopify theme** - No build tools, no package.json, no bundler. Assets are pre-compiled.
- **Shopify 2.0** - JSON templates with section-based architecture.
- **38 languages** supported with RTL (Arabic, Hebrew) via `snippets/direction.liquid`.
- **103 sections**, **97 snippets**, **22 blocks**, **51 templates**.

## Key Conventions

### Section Development
- All labels must use `t:` translation keys (never hardcoded English in schema)
- Every section must render `section-variables` snippet for colors
- Every section must render `section-spacing-style` for padding
- Standard color settings: `color_text`, `color_background`, `gradient_background`, `color_highlight`, `gradient_highlight`, `color_button`, `color_button_text`
- Standard padding settings: `padding_top`, `padding_bottom` (range 0-120, step 4, unit px)
- Use `visible_if` for conditional settings in schema
- Include `presets` with a `category` from the standard set (banners, collections, content, custom, image, newsletter, product, testimonials, text, video)
- Add `disabled_on: { "templates": ["password"] }` unless section is password-specific

### Styling
- Use CSS custom properties from `snippets/css-variables.liquid` - never hardcode colors or spacing
- Spacing scale: `--sp-0d5` through `--sp-100`
- Text scale: `--text-3xs` through `--text-8xl`
- Layout uses Tailwind-style flex utilities generated by `section-layout-class.liquid`
- Responsive scaling: values > 20px use `max(min, calc(var(--spacing-scale) * value))`

### Translations
- Storefront strings: `locales/en.default.json` with `{{ 'key' | t }}`
- Schema labels: `locales/en.default.schema.json` with `t:` prefix
- JS strings: passed via `snippets/js-variables.liquid` to global `theme` object

### File Naming
- Sections: `kebab-case.liquid` (main pages: `main-{page-type}.liquid`)
- Templates: `{type}.{variant}.json`
- Custom CSS/JS: Use `assets/custom.css` and `assets/custom.js`

## Commands

```bash
# Local development
shopify theme dev --store bio-well-eu.myshopify.com

# Pull latest
shopify theme pull --store bio-well-eu.myshopify.com

# Push changes
shopify theme push --store bio-well-eu.myshopify.com --unpublished
```
