# Styling System Reference

Complete reference for the CSS variable system, spacing, typography, and layout patterns used in the Bio-Well EU Shopify theme.

---

## CSS Custom Property Architecture

The theme uses a layered CSS variable system:

1. **Global variables** (`snippets/css-variables.liquid`) - Base colors, fonts, spacing scale, layout constants
2. **Section variables** (`snippets/section-variables.liquid`) - Per-section color overrides
3. **Inline styles** - Generated by spacing, size, gap, and border snippets

| ... | ... | ... |
| `--sp-100` | 32rem | 512px |

### Responsive Scaling

Values above a minimum threshold (default: 20px) are made responsive:

```css
/* Fixed value (below threshold) */
--padding-block-start: 16px;

/* Responsive value (above threshold) */
--padding-block-start: max(20px, calc(var(--spacing-scale) * 40px));
```

The `--spacing-scale` variable adjusts based on viewport width to proportionally scale spacing.

### Spacing Snippet Usage

```liquid
{%- render 'section-spacing-style',
  property: 'padding',       /* 'padding' or 'margin' */
  direction: 'block',        /* 'block' (top/bottom) or 'inline' (left/right) */
  start: section.settings.padding_top,
  end: section.settings.padding_bottom
-%}
```

**Generated CSS variables:**
- `--padding-block-start` / `--padding-block-end`
- `--margin-block-start` / `--margin-block-end`
- `--padding-inline-start` / `--padding-inline-end`

## Gap System

For flex/grid gap spacing between child elements.

```liquid
{%- render 'section-gap-style',
  value: section.settings.gap,
  name: 'gap'               /* CSS variable name (default: 'gap') */
-%}
```

**Generated:** `--gap: max(24px, calc(var(--spacing-scale) * 40px));`

Default minimum: 24px (vs 20px for general spacing).

### Navigation, Button, and Product Fonts

These use fluid `clamp()` sizing for responsive scaling:

```css
--font-navigation-size: clamp(0.75rem, calc(var(--navigation-size) / 16 * 1rem), 1.5rem);
--font-button-size: clamp(0.625rem, calc(var(--button-size) / 16 * 1rem), 1.25rem);
--font-product-size: clamp(0.625rem, calc(var(--product-size) / 16 * 1rem), 1.5rem);
```

## Layout System

### Page Width

| Variable | Description |
|----------|-------------|
| `--page-width` | Max container width in px |
| `--page-padding` | Responsive page horizontal padding |
| `--page-container` | `min(calc(100vw - padding*2), page-width)` |

### Responsive Breakpoints

| Breakpoint | Changes |
|------------|---------|
| < 1024px | `--page-padding: var(--sp-5)` (20px) |
| >= 1024px | `--page-padding: var(--sp-9)` (40px) |
| >= 1280px | `--page-padding: var(--sp-12)` (64px) |
| >= 1536px | `--page-padding: max(var(--sp-12), 50vw - page-width/2)` |

### Width Classes

| Class | Description |
|-------|-------------|
| `page-width` | Standard container width |
| `page-width--narrow` | Narrower container |
| (no class) | Full width |

### Layout Class Snippet

Generates Tailwind-style flex classes for responsive layouts:

```liquid
{%- render 'section-layout-class',
  content_direction: 'row',      /* 'row' or 'col' */
  horizontal_alignment: 'center', /* 'start', 'center', 'end', 'between' */
  vertical_alignment: 'center',   /* 'start', 'center', 'end', 'stretch' */
  vertical_on_mobile: true        /* Stack vertically on mobile */
-%}
```

**Output example:** `flex flex-col md:flex-row justify-center items-center flex-wrap`

## Heading Content Component

```liquid
{%- render 'heading-content',
  content: section.settings.heading,
  style: 'scribble',          /* 'scribble', 'text', 'none' */
  scribble: 'circle',         /* Icon type for scribble */
  reveal: true,               /* Fade-in animation */
  gradient: true,             /* Gradient on highlights */
  delay: 0                    /* Animation delay (x250ms) */
-%}
```

Processes `<em>` tags in the heading content to apply decorative highlight effects.

## Applying Styles: Complete Example

```liquid
<div
  id="section-{{ section.id }}"
  class="section {% if section.settings.section_width == 'page' %}page-width{% elsif section.settings.section_width == 'narrow' %}page-width--narrow{% endif %}"
  style="
    {%- render 'section-variables', section: section -%}
    {%- render 'section-spacing-style', property: 'padding', direction: 'block',
        start: section.settings.padding_top, end: section.settings.padding_bottom -%}
    {%- render 'section-gap-style', value: section.settings.gap -%}
  "
>
  {%- if section.settings.heading != blank -%}
    <{{ section.settings.heading_tag }}
      class="section__heading {{ section.settings.heading_size }} text-{{ section.settings.heading_alignment }}"
    >
      {%- render 'heading-content',
        content: section.settings.heading,
        style: section.settings.highlighted_text,
        scribble: section.settings.highlighted_scribble
      -%}
    </{{ section.settings.heading_tag }}>
  {%- endif -%}

  <div class="section__content flex {%- render 'section-layout-class',
    content_direction: section.settings.content_direction,
    horizontal_alignment: section.settings.horizontal_alignment,
    vertical_alignment: section.settings.vertical_alignment
  -%}">
    {%- for block in section.blocks -%}
      <!-- block content -->
    {%- endfor -%}
  </div>
</div>
```
